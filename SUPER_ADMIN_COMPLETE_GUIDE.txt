   - You see pending request in dashboard

   - Updates registration status to "rejected"
   - Saves rejection reason
   - Logs action in super_admin_activity
   - No account created

   - Uses email and temporary password
   - Should change password on first login
   - Gets access to admin dashboard
   - Can start accepting bookings

View Admin Performance:
   SELECT 
       admin_name,
       activity_type,
       COUNT(*) as total_actions,
       SUM(amount) as total_amount
   FROM admin_sales_activity 
   GROUP BY admin_id, admin_name, activity_type
   ORDER BY total_amount DESC;

View Today's Admin Activities:
   SELECT * FROM admin_sales_activity 
   WHERE DATE(activity_date) = CURDATE()
   ORDER BY activity_date DESC;

View Super Admin Actions:
   SELECT * FROM super_admin_activity 
   ORDER BY action_date DESC 
   LIMIT 50;
